# @capire/bookshop-java

This is our primer sample for [Getting Started in a Nutshell](https://cap.cloud.sap/docs/get-started/in-a-nutshell?impl-variant=java).


## Get it

```sh
git clone https://github.com/capire/bookshop-java
```


## Run it

```sh
mvn spring-boot:run
```

`Cmd-click` the http://localhost:8080 link in the terminal to open the app in a browser.


## Get help

- Visit the [*capire* docs](https://cap.cloud.sap) to learn about CAP.
- Especially [*Getting Started in a Nutshell*](https://cap.cloud.sap/docs/get-started/in-a-nutshell).
